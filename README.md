## 📋 Опис

Повноцінна реалізація гри «Судоку» з графічним інтерфейсом Windows Forms.  
Гравець заповнює сітку 9×9 цифрами 1–9 так, щоб кожна цифра зустрічалася рівно по одному разу у кожному рядку, стовпці та квадраті 3×3.

---

## ✨ Функціонал

| Функція | Опис |
|---|---|
| **Генерація головоломки** | Алгоритм backtracking генерує коректне поле |
| **3 рівні складності** | Легкий / Середній / Складний |
| **Підсвічування конфліктів** | Червоний колір при порушенні правил |
| **Підказки** | До 3 підказок за гру |
| **Скасування ходу (Undo)** | Повернення до попереднього стану |
| **Таймер** | Відлік часу з початку гри |
| **Таблиця рекордів** | Зберігання найкращих результатів |
| **Збереження / Завантаження** | Збереження незавершеної гри у файл `.sdk` |

## 🚀 Запуск

### Вимоги
- Windows 10/11
- [.NET 6 SDK](https://dotnet.microsoft.com/download/dotnet/6.0)
- Visual Studio 2022 (або `dotnet CLI`)

### Через Visual Studio
1. Відкрити `SudokuGame.sln`
2. Натиснути `F5` або кнопку «Запустити»

### Через командний рядок
```bash
cd SudokuGame
dotnet run
```

## 🎮 Як грати

1. Оберіть рівень складності у випадаючому списку
2. Натисніть **▶ Нова гра**
3. Клікніть на порожню клітинку та введіть цифру від 1 до 9
4. Червоний колір = конфлікт з правилами
5. Використовуйте **💡 Підказку** (до 3 разів) або **↩ Скасувати** для виправлення помилок
6. Заповніть всі клітинки без конфліктів — перемога!

## 📁 Формат файлу збереження (`.sdk`)

```
0              ← рівень складності (0=Easy, 1=Medium, 2=Hard)
5,3,0,0,7,...  ← рядки поточного стану (9 рядків)
5,3,4,6,7,...  ← рядки розв'язку (9 рядків)
0;1            ← координати початкових клітинок (рядок;стовпець)
...
```
